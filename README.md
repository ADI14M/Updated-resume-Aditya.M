## 🚀 About

A modern, interactive 3D portfolio website showcasing my work in **AI/ML**, **Data Engineering**, and **Full-Stack Development**. Features a fully animated 3D character, immersive physics-based tech stack visualization, smooth scroll animations, and a custom cursor — all built for performance and visual impact.

## 🛠️ Tech Stack

| Category | Technologies |
|----------|-------------|
| **Frontend** | React, TypeScript, HTML5, CSS3 |

## ⚡ Features

- 🎭 **Animated 3D Character** — Interactive WebGL model with bone-based animations
- 🔮 **Physics-Based Tech Stack** — Floating 3D spheres with real-time physics simulation
- 🎯 **Custom Cursor** — Context-aware animated cursor that reacts to elements
- 🌊 **Smooth Scrolling** — GSAP ScrollSmoother for buttery-smooth navigation
- 📱 **Responsive Design** — Optimized for all screen sizes
- ⚡ **Loading Screen** — Animated loading sequence with progress tracking
- 🎨 **Project Carousel** — Swipeable showcase of featured projects

## 🏃 Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/ADI14M/Updated-resume-Aditya.M.git

# Navigate to the project
cd Updated-resume-Aditya.M

# Install dependencies
npm install

# Start the development server
npm run dev
```

> **Note:** This project uses GSAP trial plugins for ScrollSmoother. For production hosting, you'll need GSAP Club plugins. See: https://gsap.com/docs/v3/Installation/
